# samarinda-project

Deteksi sawit

Step Menjalankan Scripts

1. Membuat venv

```powershell
 python -m vevn venv
```

2. Aktivasi venv

```powershell
.\venv\Scripts\activate
```

3. Install Requirements

```powershell
pip install samarinda.txt
```

4. Jalankan Script

```powershell
python main.py
```
